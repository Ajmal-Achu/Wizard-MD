### FOR TERMUX USER
1. Type below given commands one by one in Termux.
```
apt update && apt upgrade
apt install bash
apt install libwebp
apt install git-y
apt install nodejs -y
apt install ffmpeg -y
apt install wget
apt install imagemagick -y
git clone https://github.com/Ajmal-Achu/Wizard-MD
cd Wizard-MD
rm -r session.json
node.
```
